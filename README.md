# Expense Tracker

A Java-based Expense Tracker application developed using Java Swing and MySQL.

## Features
- Add Expenses
- View Expenses
- Update Expenses
- Delete Expenses
- Search Expenses
- Calculate Total Expenses

## Technologies Used
- Java
- Java Swing
- MySQL
- JDBC

- src/
- lib/
